# IRIS_DATASET_VISUALIZATION_DH

## Objective
Understand how to read, summarize, and visualize a dataset.

## Approach
● Display dataset structure using .shape, .columns, and .head().
● Create basic visualizations:
○ Scatter plot to analyze relationships between variables.
○ Histogram to examine data distribution.
○ Box plot to detect outliers and spread of values.s

## Results and Insights
- The dataset contains 150 samples with 3 flower species
- Petal length and width show clear separation between species
- Visualization helped identify patterns and distributions in data
- Iris-setosa is easily separable from other species
